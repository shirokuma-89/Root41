# 2020 版 仕様

## 入力系統

| デバイス               | 個数 | ポート        |
| ---------------------- | ---- | ------------- |
| ボールセンサ           | x13  | A0 - A15,D26  |
| ラインセンサ           | x20  | D30 - D49     |
| ジャイロセンサ(GY-521) | x1   | I2C           |
| カメラ(Pixy)           | x1   | I2C,SPI,UART  |
| 捕捉センサ             | x1   | D29           |
| タクトスイッチ         | x3   | D22 - D23,D25 |

## 出力系統

| デバイス         | 個数 | ポート                         |
| ---------------- | ---- | ------------------------------ |
| モーター         | x3   | D4 - D6,D7 - D9,D10 - D12      |
| ブザー           | x1   | ATtiny85（I2C,デジタル入出力） |
| NeoPixel         | x16  | D28                            |
| ソレノイド(予備) | x1   | D27                            |
